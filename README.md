# 年度进度条工具 | Year Progress Bar

> 可视化年度进度，珍惜每一刻时光

一个美观、功能强大的年度进度条Web工具，帮助你可视化时间进度，保持动力和专注。

## ✨ 功能特性

### 核心功能
- 🎯 **多种进度类型**: 年度、月度、周进度，甚至人生进度
- 🎨 **多种进度条形状**: 线性、圆形、弧形三种可视化方式
- 🎭 **完全自定义**: 颜色、主题、显示选项全部可配置
- 🌓 **暗黑模式**: 支持浅色/深色主题切换

### 高级功能
- 🔔 **智能通知**: 里程碑提醒、每日/每周进度通知
- 📊 **历史分析**: 记录和查看历史进度数据
- 📤 **社交分享**: 一键分享到 Twitter、微博等平台
- 🔗 **嵌入功能**: 生成代码嵌入到你的网站
- 💾 **本地存储**: 所有数据保存在本地浏览器，隐私安全

### 技术亮点
- ⚡️ **现代化技术栈**: Next.js 15 + React 18 + TypeScript
- 🎨 **精美UI**: Tailwind CSS + Framer Motion 动画
- 📱 **响应式设计**: 完美适配桌面和移动设备
- 🚀 **快速加载**: 静态导出，秒开体验

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

在浏览器中打开 [http://localhost:3000](http://localhost:3000)

### 构建生产版本

```bash
npm run build
npm start
```

### 静态导出（可选）

如果需要导出为静态网站：

1. 修改 `next.config.js`:

```javascript
module.exports = {
  output: 'export',
  images: {
    unoptimized: true
  }
}
```

2. 构建静态文件:

```bash
npm run build
```

静态文件将生成在 `out` 目录。

## 📖 使用指南

### 基础使用

1. 选择进度类型（年度、月度、周、人生）
2. 选择进度条形状（线性、圆形、弧形）
3. 自定义颜色和显示选项
4. 查看实时进度和激励消息

### 高级功能

#### 启用通知
1. 点击"启用通知"按钮
2. 允许浏览器通知权限
3. 选择提醒频率（里程碑、每日、每周）
4. 及时接收进度更新

#### 记录历史
1. 在"历史数据与趋势"面板
2. 点击"记录当前进度"
3. 查看历史趋势和统计

#### 分享进度
1. 点击"分享与导出"
2. 选择分享平台或复制文本
3. 或复制嵌入代码到你的网站

## 🎨 自定义主题

应用提供了多个预设颜色主题：
- 蓝色（默认）
- 紫色
- 绿色
- 红色
- 橙色
- 青色

也可以自定义主色调和背景色，创造个人专属风格。

## 📊 数据隐私

- 所有配置和历史数据保存在浏览器的 `localStorage` 中
- 不会上传任何数据到服务器
- 可以随时清空本地数据
- 完全隐私保护
